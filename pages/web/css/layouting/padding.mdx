# Padding

`padding` property sets the padding area of an element (the distance between the element's content and its border).

<Image path="/res/web/css/box-model.png" caption="Illustration of CSS Box model" />

## Common Padding Properties

### `padding`

Specifying `padding` allows you to set the top, right, bottom, and left padding of an element (remember: clockwise direction).

If you specify less than 4 padding lengths, the missing lengths would be inferred from the lengths that you specified (it will try to make symmetrical padding).

Possible length values:

### `padding-top`

Sets the element's top padding. (See `padding`.)

### `padding-right`

Sets the element's right padding. (See `padding`.)

### `padding-bottom`

Sets the element's bottom padding. (See `padding`.)

### `padding-left`

Sets the element's left padding. (See `padding`.)

## Obscured Padding Properties

### `padding-inline-start`

Sets the padding at the start of "inline" direction. If you're writing horizontally from left to right, then it maps to `padding-left`.

(Where this direction is depends on `writing-mode`, `direction`, and `text-orientation`. See `padding` for the list of possible values.)

### `padding-inline-end`

Sets the padding at the end of "inline" direction. If you're writing horizontally from left to right, then it maps to `padding-right`.

(The "inline" direction depends on `writing-mode`, `direction`, and `text-orientation`. See `padding` for the list of possible values.)

### `padding-inline`

Shorthand for `padding-inline-start` and `padding-inline-end`.

```css
p {
  padding-inline: <padding-inline-start> <padding-inline-end>;
  padding-inline: <padding-inline-start>;
}
```

### `padding-block-start`

Sets the padding at the start of "block" direction (perpendicular to "inline" direction). If you're writing horizontally from left to right, top to bottom, then it maps to `padding-top`.

(Where this direction is depends on `writing-mode`, `direction`, and `text-orientation`. See `padding` for the list of possible values.)

### `padding-block-end`

Sets the padding at the end of "block" direction (perpendicular to "inline" direction). If you're writing horizontally from left to right, top to bottom, then it maps to `padding-bottom`.

(Where this direction is depends on `writing-mode`, `direction`, and `text-orientation`. See `padding` for the list of possible values.)

### `padding-block`

Shorthand for `padding-block-start` and `padding-block-end`.

```css
div {
  padding-block: <padding-block-start> <padding-block-end>;
  padding-block: <padding-block-start>;
}
```
