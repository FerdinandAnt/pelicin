# Image

The image data type represents a two-dimensional image (could be loaded from a URL, or a gradient).

## URL

You can use `url(...)`.

```css
div {
  background-image: url('/path/to/picture.png');
}
```

## Gradient

### Linear Gradient

Transition colors progressively along an imaginary line.

- **<code>linear-gradient(<var>angle</var>, <var>...stops</var>)</code>**
- **<code>repeating-linear-gradient(<var>angle</var>, <var>...stops</var>)</code>**

**Note:**

- _`angle`_ (optional) could be keywords like `to top`, `to top left`, or in degrees like `180deg`.
  (0 deg = 12 o'clock, remember "clockwise")

- _`...stops`_ would be comma-separated list of colors, each one optionally followed by a percentage or a length
  (e.g. `red`, `red 15px`, `red 0%`, `red 0% 50%`).

### Radial Gradient

Transition colors progressively from a center point (origin)

- **<code>radial-gradient(<var>config</var>, <var>...stops</var>)</code>**
- **<code>repeating-radial-gradient(<var>config</var>, <var>...stops</var>)</code>**

**Note:**

- _`config`_ (optional) would determine (1) the ending shape, (2) the size, and (3) the origin (center-point) of the shape.
  Separate the values with space.

  - The ending shape could be `ellipse` or `circle`

  - The size parameter could be `closest-side`, `closest-corner`, `farthest-side`, or `farthest-corner` (determines how far the shape extends from the orrigin)

  - The origin should be specified with lengths or percentages, e.g. `at 10px 10px` or `at 50% 50%`

- _`...stops`_ would be comma-separated list of colors, each one optionally followed by a percentage or a length
  (e.g. `red`, `red 15px`, `red 0%`, `red 0% 50%`).

<WebEditor
  hideHTML
  html={smartTrim(`

    </div>

    }
  `)}
/>

### Conic Gradient

- **<code>conic-gradient(<var>config</var>, <var>...stops</var>)</code>**

**Note**

- _`config`_ (optional) would determine (1) the starting degree and (2) the origin (center-point) of the shape.
  Separate the values with space.

  - The starting degree would be something like `from 20deg` (0 deg = 12 o'clock, remember "clockwise")

  - The origin should be specified with lengths or percentages, e.g. `at 10px 10px` or `at 50% 50%`

- _`...stops`_ would be comma-separated list of colors, each one optionally followed by a percentage or a length
  (e.g. `red`, `red 15px`, `red 0%`, `red 0% 50%`).
