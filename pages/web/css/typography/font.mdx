# Font

This page lists the CSS properties to customize an element's font.

## Font At-Rules

### `@font-face`

Defines a custom font.

**Required properties:**

<!-- prettier-ignore -->
```css
@font-face {
  font-family: examplefont;
  src: local(Example Font),
    url('examplefont.woff') format('woff'),
    url('examplefont.otf') format('opentype');
}
```

## Font Properties

### `font`

Shorthand for the following properties (in order). `font-size` and `font-family` are required.

- `font-style`
- `font-variant`
- `font-weight`
- `font-stretch`
- `font-size`
- `line-height`
- `font-family`
