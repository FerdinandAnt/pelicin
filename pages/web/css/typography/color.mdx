# Color

`color` property sets text color.

For other properties, you can use the `currentcolor` keyword to use the current text `color`.

## CSS Property

### `color`

Sets the text (foreground) color.

<WebEditor
  html={smartTrim(`
    <div id="color-example">
      Hi!
    </div>    
  `)}
  css={smartTrim(`
    #color-example {
      color: blue;
      padding: 10px;
      border: 4px solid currentcolor;
    }    
  `)}
/>
