# Spacing and Alignment

This page lists CSS properties which relates to spacing and alignment of text

## Text Spacing Properties

### `letter-spacing`

Adds/subtracts the horizontal spacing between letters. Specify `normal` (auto) or a length (e.g. `0.2em` or `-0.2em`).

<WebEditor
  html={smartTrim(`
    <div id="letter-spacing-example">
      Lorem ipsum dolor sit amet
    </div>    
  `)}
  css={smartTrim(`
    #letter-spacing-example {
      letter-spacing: 0.2em;
    }    
  `)}
/>

### `word-spacing`

Adds/subtracts the horizontal spacing between words. Specify `normal` (auto) or a length (e.g. `0.2em` or `-0.4ch`).

### `line-height`

Sets the height of the line box (distance between the lines of the text). Specify:

### `text-indent`

Sets the indentation at the beginning of paragraphs.

Specify a length (e.g. `2em` or `-2em`) or percentage (relative to the parent's width).

  html={smartTrim(`
