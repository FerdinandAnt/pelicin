# Other Text Properties

CSS text properties that don't belong elsewhere are listed here.

## CSS Properties

### `text-transform`

Transforms an element's text (usually to change its capitalization).

  items={[

### `text-rendering`

Hints what to optimize for when rendering text.

<DocumentationList
  items={[
    {
      name: 'auto',
      description: 'let the browser decide (default)',
    },
    {
      name: 'optimizeSpeed',
      description: 'optimize for speed (disables kerning and ligatures)',
    },
    {
      name: 'optimizeLegibility',
      description: 'optimize for legibility (enables kerning and ligatures)',
    },
    {
      name: 'geometricPrecision',
      description:
        'optimize for geometric precision (may make some fonts look good when scaled to non-integer sizes)',
    },
  ]}
/>
