# Display

The `display` element sets whether an element is "block" or "inline".

## Possible Values

<DocumentationList
  label="Block/inline"
  items={[
    {
      name: 'block',
      description: (
        <>
          Generates a block element (generates line breaks before and after the element in the
          normal flow
        </>
      ),
    },
    {
      name: 'inline',
      description: <>Generates an inline element (flows on the same line)</>,
    },
    {
      name: 'inline-block',
      description: <>Generates a block element that flows on the same line</>,
    },
  ]}
/>

<DocumentationList
  label="None"
  items={[
    {
      name: 'none',
      description: (
        <>
          Does not render the element (the element would also take no space on the screen, as if it
          wasn't there)
        </>
      ),
    },
  ]}
/>
