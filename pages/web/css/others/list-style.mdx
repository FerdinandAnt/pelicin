# List Style

This page lists the CSS properties related to styling a list.

To select the list/marker pseudo-element itself, use [`::marker`](http://localhost:3000/web/css/selectors/pseudo-elements#::marker).

## CSS Properties

### `list-style`

Shorthand for `list-style-type`, `list-style-image`, `list-style-position`

```css
ul {
  list-style: inside;
  list-style: georgian inside;
  /* <list-style-type> <list-style-image> <list-style-position> */
  list-style: lower-roman url('../img/shape.png') outside;
}
```

### `list-style-type`

Sets the item marker to use (disc, character, etc.).
The color of the marker will be the same as the color of the element it applies to.

Specify ([see the complete list-style-type here](https://developer.mozilla.org/en-US/docs/Web/CSS/list-style-type)):

### `list-style-image`

Sets the image to be used as item marker.

```css
ul {
  list-style-image: url('starsolid.gif');
}
```

### `list-style-position`

Sets the position of the `::marker` outside or inside of the list item.

Specify `outside` (default) or `inside`.
