# Introduction

CSS (Cascading Style Sheets) allows you to _style_ HTML elements, defining how they should look like.

## CSS Syntax

Each CSS rule has the following:

- one or more <dfn>selectors</dfn> (these indicate which HTML elements should be affected by the rule)
- one or more <dfn>declarations</dfn> in the form of <dfn>property-value pairs</dfn> (these define what the rule does)

<Image
  path="/res/web/css/css-rule.png"
  caption="Basic CSS syntax: selector, property, and value."
/>

Anything between `/*` and `*/` are comments and do nothing.

## Example

<WebEditor
  html={smartTrim([
    `
      <!-- Style -->
      <style>
        /* Make all <b> with class "example" red and 24px */
        b.example {
          color: red;
          font-size: 24px;
        }
        /* Make all <em> and <u> with class "example" green */
        i.example, u.example {
          color: green;
        }
      </style>
    `,
    `
      <!-- Content -->
      <b class="example">one</b>
      <i class="example">two</i>
      <u class="example">three</u>
    `,
  ])}
/>
