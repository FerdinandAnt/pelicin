## `<ul>`

Creates <dfn>unordered list</dfn> (unnumbered).

```html
<ul>
  <li>Apple</li>
  <li>Banana</li>
  <li>Orange</li>
</ul>
```

    <li>Apple</li>
    <li>Banana</li>
    <li>Orange</li>
  </ul>
</LightBox>

## `<ol>`

Creates <dfn>ordered list</dfn> (numbered).

<Badge color="blue">Possible attributes</Badge>
<ul>
  <li>
    <Attribute>type=[...]</Attribute> &mdash; numbering type. Can be:
    <ul>
      <li>
        <code>1</code> <i>(numbers: 1, 2, 3, ...)</i>
      </li>
      <li>
        <code>a</code> <i>(lowercase letters: a, b, c, ...)</i>
      </li>
      <li>
        <code>A</code> <i>(uppercase letters: A, B, C, ...)</i>
      </li>
      <li>
        <code>i</code> <i>(lowercase Roman: i, ii, iii, ...)</i>
      </li>
      <li>
        <code>I</code> <i>(uppercase Roman: I, II, III, ...)</i>
      </li>
    </ul>
  </li>
  <li>
    <Attribute>start=[...]</Attribute> &mdash; from what number the list starts (always use arabic
    numeral like 1, 2, 3 for the value regardless of <code>type</code>)
  </li>
  <li>
    <Attribute>reversed</Attribute> &mdash; start numbering in reverse order
  </li>
</ul>

```html
<ol>
  <li>One</li>
  <li>Two</li>
  <li>Three</li>
</ol>
```

<LightBox label="Preview">
  <ol>
    <li>One</li>
    <li>Two</li>
    <li>Three</li>
  </ol>
</LightBox>

## `<li>`

Defines <dfn>list item</dfn>. It must be put inside `<ol>` or `<ul>`.

You can create nested list by putting `<ol>` or `<ul>` inside `<li>`.

<Badge color="blue">Possible attributes</Badge>
<ul>
  <li>
    <Attribute>value=[...]</Attribute> &mdash; set current numeral value for <code>&lt;ol&gt;</code>{' '}
    (always use arabic numeral like 1, 2, 3 for the value). The next list item will continue
    numbering from this value.
  </li>
</ul>

```html
<ol type="I">
  <li value="3">third item</li>
  <li>fourth item</li>
  <li>
    fifth item
    <ul>
      <li>subitem A</li>
      <li>subitem B</li>
    </ul>
  </li>
</ol>
```
