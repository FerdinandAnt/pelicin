## `<svg>`

Used to draw graphics using <a href="https://developer.mozilla.org/en-US/docs/Web/SVG/Element/svg">SVG</a>.

An `<svg>` tag defines a new coordinate system and viewport.

<NoteBox type="info">
  <b>NOTE:</b> The <Code str="xmlns" /> attribute is only required on the outermost svg element of
  SVG documents. It is unnecessary for inner svg elements or inside HTML documents.
</NoteBox>
