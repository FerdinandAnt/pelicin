## `<blockquote>`

Represents a quotation block.

<DocumentationList
  label="Possible attributes"
  items={[{ name: 'cite=[...]', description: 'URL to the source of quotation' }]}
/>

```html
<blockquote>
  Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut
  labore et dolore magna aliqua.
</blockquote>
```

## `<pre>`

Represents a <dfn>preformatted text</dfn>.

The text will be displayed as typed in the HTML file.

```html
<pre>
Roses are red,
  Violets are blue,
Sugar is sweet,
  And so are you. 
</pre>
```

## `<dl>`, `<dt>`, and `<dd>`

`<dl>` represents a <dfn>definition list</dfn>.

`<dt>` represents a <dfn>definition term</dfn> in a `<dl>`.

`<dd>` represents a <dfn>definition description</dfn> in a `<dl>`.

```html
<!-- source: https://developer.mozilla.org/en-US/docs/Web/HTML/Element/dl -->
<dl>
  <dt>Beast of Bodmin</dt>
  <dd>A large feline inhabiting Bodmin Moor.</dd>

  <dt>Morgawr</dt>
  <dd>A sea serpent.</dd>

  <dt>Owlman</dt>
  <dd>A giant owl-like creature.</dd>
</dl>
```

    <dd>A sea serpent.</dd>

    <dd>A giant owl-like creature.</dd>
  </dl>
</LightBox>

## `<figure>` and `<figcaption>`

`<figure>` represents a self-contained content.

`<figcaption>` represents caption for `<figure>`.

<!-- prettier-ignore -->
```html
<figure>
    <img src="/res/web/html/lucas-benjamin-unsplash.jpg" alt="An abstract photo"/>
    <figcaption>A photo by Lucas Benjamin - Unsplash</figcaption>
  </figure>
```
