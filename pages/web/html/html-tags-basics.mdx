## `<div>` and `<span>`

`<div>` is used to create generic <dfn>block-level container</dfn>, which means the content inside it will take the whole line.

`<span>` is used to create generic <dfn>inline container</dfn>, which means the content inside it will flow with the current line of text.

```html
<!-- div creates its own line -->
<div style="background-color: red; color: white">
  I'm creating a new line!
</div>

<!-- span flows inline with the surrounding text -->
Lorem ipsum
<span style="background-color: blue; color: white">
  I'm flowing with the text
</span>
dolor sit amet
```

  Lorem ipsum <span style={{ backgroundColor: 'blue', color: 'white' }}>
    I'm flowing with the text
  </span> dolor sit amet
</LightBox>

## `<h1>` &ndash; `<h6>`

These are used to create <em>headers</em>. They are block-level elements.

- `<h1>` would normally be used for title
- `<h2>` would be for the subtitle
- `<h3>` would be for the sub-subtitle
- so on and so forth.

```html
<h1>H1: Nice title</h1>
<h2>H2: Nice subtitle</h2>
<h3>H3: Nice boat</h3>
```

  <h3 style={{ border: 'none', margin: 0, padding: 0 }}>Nice boat</h3>

## `<p>`

Marks a paragraph. It's a block-level element.

<!-- prettier-ignore -->
```html
<p>Lorem ipsum dolor sit amet. Consectetur adipisicing elit.</p>
<p>Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.</p>
```

## `<a>`

Creates <dfn>anchors</dfn> and hyperlinks.

<Badge color="blue">Possible attributes</Badge>
<ul>
  <li>
    <Attribute>href</Attribute> &mdash; where the link points to
  </li>
  <li>
    <Attribute>target</Attribute> &mdash; where to open it (
    <a href="/web/html/html-attributes-recurring#target">read here</a>)
  </li>
  <li>
    <Attribute>download</Attribute> &mdash; instead of opening the link, prompt the user to download
    it. If value is given, it will be the name of the downloaded file. (Only works for same-origin
    URLs, or <code>blob:</code> and <code>data:</code> schemes)
  </li>
</ul>

```html
<p>To search the internet, <a href="https://google.com">click here</a>!</p>

<!-- Prompts download when clicked -->
<p><a href="/site/skulls.png" download>Download an image?</a></p>

<!-- When clicked, the header will be brought to top of page -->
<h1 id="my-anchor">My Anchor</h1>
<p>Click <a href="#my-anchor">here</a> to go to My Anchor</p>
```

  <p>Click <a href="#my-anchor">here</a> to go to my-anchor</p>
