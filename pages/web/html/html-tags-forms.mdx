## `<form>`

Represents a <em>form</em>.

<Badge color="blue">Attributes related to submission</Badge>
<ul>
  <li>
    <Attribute>action=[...]</Attribute> &mdash; the URL to which the form is submitted (which URL
    performs some action on the form data)
  </li>
  <li>
    <Attribute>method=[...]</Attribute> &mdash; the HTTP method with which the form is submitted;
    specify either <code>post</code> or <code>get</code>
  </li>
  <li>
    <Attribute>enctype=[...]</Attribute> &mdash; if <code>method="post"</code>, this specifies the
    MIME type of the form submission. Can be:
    <ul>
      <li>
        <code>application/x-www-form-urlencoded</code> <i>(the default value)</i>
      </li>
      <li>
        <code>multipart/form-data</code>{' '}
        <i>
          (use if there is any <code>{`<input type="file">`}</code>)
        </i>
      </li>
    </ul>
  </li>
  <li>
    <Attribute>novalidate</Attribute> &mdash; don't validate form before being submitted
  </li>
  <li>
    <Attribute>target=[...]</Attribute> &mdash; where to display the response (
    <a href="/web/html/html-attributes-recurring#target">read here</a>)
  </li>
</ul>

<Badge color="blue">Other attributes</Badge>
<ul>
  <li>
    <Attribute>accept-charset=[...]</Attribute> &mdash; space-separated character encodings the
    server accepts, e.g. <code>utf-8</code>
  </li>
  <li>
    <Attribute>autocomplete=[...]</Attribute> &mdash; indicates whether input fields can be
    autocompleted by the browser; specify either <code>on</code> or <code>off</code>
  </li>
  <li>
    <Attribute>name=[...]</Attribute> &mdash; name of the form (must be unique)
  </li>
  <li>
    <Attribute>rel=[...]</Attribute> &mdash;{' '}
    <a href="/web/html/html-attributes-recurring#rel">read here</a>
  </li>
</ul>

<!-- prettier-ignore -->
```html
<form method="post" action="/login">
  Username: <input type="text"><br>
  Password: <input type="password"><br>
  <input type="submit" value="Login">
</form>
```

## `<input>`

Creates interactive form input.

<Badge color="blue">Common attributes (apply to most, if not all)</Badge>
<ul>
  <li>
    <Attribute>name=[...]</Attribute> &mdash; name of this input
  </li>
  <li>
    <Attribute>value=[...]</Attribute> &mdash; value of this input
  </li>
  <li>
    <Attribute>type=[...]</Attribute> &mdash; type of this input, read next section
  </li>
  <li>
    <Attribute>autofocus</Attribute> &mdash; sets focus on input when the page has finished loading
  </li>
  <li>
    <Attribute>disabled</Attribute> &mdash; disables input (the user cannot interact with it)
  </li>
  <li>
    <Attribute>readonly</Attribute> &mdash; makes the user unable to change this input's value
  </li>
  <li>
    <Attribute>required</Attribute> &mdash; forces the user to specify this input's value before
    submitting the form
  </li>
  <li>
    <Attribute>autocomplete=[...]</Attribute> &mdash; hints if the browser can autofill this input;
    specify either <code>on</code> or <code>off</code>
  </li>
  <li>
    <Attribute>form=[...]</Attribute> &mdash; associates this input to an <code>{`<form id="...">`}</code>{' '}
    element with the given <code>id</code> (allows you to place the input anywhere in the document)
  </li>
  <li>
    <Attribute>list=[...]</Attribute> &mdash; associates this input to an <code>{`<datalist id="...">`}</code>{' '}
    element with the given <code>id</code>.
  </li>
</ul>
